# 游戏引擎基础 {#game-engine-fundamentals}

本章介绍了使用Bevy作为游戏引擎的基础知识。<br/>
您应该熟悉Bevy编程。为此，请参阅Bevy编程框架章节。<br/>
本章涵盖的主题适用于所有想要使用Bevy的项目，而不仅仅是一个ECS库。如果您正在制作游戏或使用Bevy的其他应用程序，这适用于您。<br/>
本章仅涵盖一般基础知识。更复杂的主题将在本书的其他章节中单独介绍：<br/>
-   输入处理
-   窗口管理
-   资产管理
-   一般图形特性
-   2D图形
-   3D图形
-   音频
-   Bevy UI框架
